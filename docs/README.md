# MPCP Documentation

**Machine Payment Control Protocol** — a cryptographically enforced pipeline for autonomous and software-controlled payments.

This is the public documentation site for MPCP. The docs can be served as static files (e.g., with MkDocs, VitePress, or GitHub Pages) or read directly as Markdown.
